# Conocimiento Base - Xavi

Mi base de conocimiento compartida con Claude para todos mis proyectos.

## Como Usar

Cuando abras Claude en cualquier proyecto, dile:

```
lee conocimiento-base/CONOCIMIENTO.md
```

Claude sabra:
- Quien eres y como trabajas
- Tus preferencias tecnicas
- Decisiones que ya tomaste
- Problemas que ya resolviste

## Archivos

| Archivo | Contenido |
|---------|-----------|
| `CONOCIMIENTO.md` | Preferencias, decisiones, stack |
| `SNIPPETS/` | Codigo reutilizable |
| `ERRORES/` | Problemas resueltos |

## Autoguardado

Se guarda automaticamente cada 5 minutos si configuras el Task Scheduler.
